# Multi-Timer-Dashboard

## Installation
This is a Multi Timer Dashboard where you can add multiple timers, each with a custom name and duration for example hours, minutes, seconds.
You can independently Start/Pause, Reset, or Delete each timer.
All timers run simultaneously, automatically counting down and updating on the screen every second.
